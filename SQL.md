## Create the Empty Table 
```sql
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype,
   ....
);
```
Example : 
```sql
CREATE TABLE Summary (
    ID int,
    class varchar(255),
    price int,
    quantity int
)
```

> | ID | class | price | quantity |
> | ---------- | ----------- | ---------- | ----------- |
